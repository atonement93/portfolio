# 엠카운트다운 개발 프로젝트



  ### :loudspeaker: 개요
  


* #### 프로젝트 설명


    학습을 목적으로 구현하기에 적합한 엠카운트다운 웹 사이트를 클론코딩하였다. 
    
    
    UI가 비교적 깔끔하며, 백엔드의 핵심적인 기능으로 구현할 게시판과 댓글 기능을 클론코딩 해볼 수 있는 좋은 표본으로 판단되어 엠카운트 웹 사이트를 선택하게 되었다.
